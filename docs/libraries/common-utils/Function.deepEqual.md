# Function: deepEqual()

> **deepEqual**(`value1`, `value2`): `boolean`

Hàm helper để so sánh sâu hai giá trị (bao gồm object và array).

## Parameters

### value1

`unknown`

Giá trị thứ nhất.

### value2

`unknown`

Giá trị thứ hai.

## Returns

`boolean`

True nếu hai giá trị bằng nhau về mặt cấu trúc và giá trị, ngược lại là False.
