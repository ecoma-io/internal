# Function: getRootDomain()

> **getRootDomain**(`hostname`): `string`

Lấy root domain từ hostname.

## Parameters

### hostname

`string`

Tên host (ví dụ: sub.example.com)

## Returns

`string`

Root domain (ví dụ: example.com)

## Example

```ts
getRootDomain('sub.example.com'); // 'example.com'
```
