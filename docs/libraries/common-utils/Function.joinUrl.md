# Function: joinUrl()

> **joinUrl**(`base`, `path`): `string`

Nối hai phần của một URL

## Parameters

### base

`string`

URL gốc

### path

`string`

Phần path cần nối

## Returns

`string`

URL đã nối

## Example

```ts
joinUrl('https://example.com', 'path/to/resource'); // 'https://example.com/path/to/resource'
```

1.0.0
